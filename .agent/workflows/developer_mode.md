# 💻 개발자 모드 (Developer Mode)

이 워크플로우는 기획된 내용과 디자인 가이드를 바탕으로 실제 동작하는 소프트웨어를 개발하는 과정을 돕습니다.

1. **개발 환경 점검**
   - 현재 프로젝트 디렉토리 구조를 확인합니다.
   - `package.json` 등 설정 파일이 올바른지 점검합니다.
   - 필요한 라이브러리나 의존성(Dependency)이 설치되어 있는지 확인합니다.

2. **아키텍처 및 구현 계획 수립 (Implementation Plan)**
   - 기획 문서(`docs/planning/requirements.md`)가 있다면 내용을 숙지합니다.
   - "어떤 순서로 개발할 것인가?"에 대한 로드맵을 사용자에게 제안합니다.
     - 예: 1단계 - 기본 레이아웃 구성, 2단계 - 메인 페이지 기능 구현, 3단계 - API 연동
   - 폴더 구조(Folder Structure)를 제안하고, 필요시 생성합니다.

3. **기능 구현 (Coding)**
   - 계획에 따라 실제 코드를 작성합니다. React, Vue, Python 등 프로젝트 언어에 맞춰 최적의 코드를 작성합니다.
   - **스타일 적용**: 디자인 가이드(`docs/design/style_guide.md`)가 있다면 이를 엄격히 준수하여 CSS/Tailwind 등을 적용합니다.
   - **클린 코드**: 변수명, 주석, 함수 분리 등을 신경 써서 유지보수가 쉬운 코드를 작성합니다.

4. **테스트 및 디버깅 (Testing & Debugging)**
   - 작성한 코드가 의도대로 작동하는지 검증합니다.
   - 오류가 발생하면 원인을 분석하고 해결책을 제시합니다.
   - 필요시 단위 테스트(Unit Test) 코드를 작성하여 안정성을 높입니다.

5. **문서화 (Documentation)**
   - 개발이 완료된 기능에 대해 `README.md`를 업데이트합니다.
   - 설치 방법, 실행 방법, 주요 기능 설명을 추가합니다.
