# 📢 마케터 모드 (Marketer Mode)

이 워크플로우는 서비스의 가치를 효과적으로 알리고 사용자를 유치하기 위한 마케팅 전략을 수립하는 과정을 돕습니다.

1. **마케팅 폴더 준비**
   - 프로젝트 루트에 `docs/marketing` 폴더가 있는지 확인하고, 없으면 생성합니다.
   - (참고: `run_command`를 사용하여 `mkdir -p docs/marketing`을 실행할 수 있습니다.)

2. **USP(고유 강점) 도출 인터뷰**
   - 사용자에게 이 서비스만의 차별점이 무엇인지 물어봅니다.
   - 경쟁사와 비교했을 때 어떤 점이 좋은지, 사용자가 왜 이 서비스를 써야 하는지를 파악합니다.
   - 이를 바탕으로 핵심 메시지(Core Message)를 정의합니다.

3. **마케팅 전략 수립 (Strategy)**
   - 인터뷰 내용을 바탕으로 `docs/marketing/strategy.md` 파일을 생성하거나 업데이트합니다.
   - 타겟 고객에 맞는 홍보 채널(블로그, SNS, 커뮤니티 등)을 추천합니다.
   - 런칭 전/후 단계별 마케팅 로드맵(Roadmap)을 제시합니다.

4. **콘텐츠 기획 및 카피라이팅 (Copywriting)**
   - 서비스 소개 문구, 슬로건, SNS 게시글 등 구체적인 콘텐츠를 작성합니다.
   - 필요 시 `generate_image` 도구를 활용하여 광고용 이미지 시안을 제안합니다.
   - 검색 엔진 최적화(SEO)를 고려한 키워드 전략을 문서에 포함합니다.

5. **실행 계획 제안**
   - 언제, 어디에, 무엇을 홍보할지에 대한 구체적인 액션 아이템(Action Item)을 정리합니다.
   - 예: "1주차 - 티저 페이지 공개 및 사전 예약 받기", "2주차 - 기능 소개 카드뉴스 배포" 등
